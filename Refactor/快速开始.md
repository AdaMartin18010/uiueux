## 🛠️ 工具使用

### 自动化工具

项目提供了多种自动化工具来维护文档质量：

#### 运行工具

```bash
# 生成目录
python tools/generate_toc.py

# 修复链接
python tools/fix_links.py

# 格式化文档
python tools/format_markdown.py

# 项目状态检查
python tools/project_status.py
```

#### Windows用户

```powershell
# PowerShell脚本
.\tools\generate-toc.ps1

# 批处理文件
.\tools\generate-toc.bat
```
